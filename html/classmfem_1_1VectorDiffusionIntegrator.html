Detailed Description

Integrator for (Q grad u, grad v) = sum_i (Q grad u_i, grad v_i) e_i e_i^T for FE spaces defined by 'dim' copies of a scalar FE space. Where e_i is the unit vector in the i-th direction. The resulting local element matrix is a block-diagonal matrix consisting of 'dim' copies of a scalar diffusion matrix in each diagonal block.
